# ♟️ Chess Game

A browser-based two-player chess game built with vanilla HTML, CSS, and JavaScript. Play chess directly in your browser with no installations or dependencies required.

## Features ✨

- ♟️ Full two-player turn-based gameplay (White goes first)
- 🟢 Visual move indicators — valid moves highlighted with circles
- ⚔️ Piece capturing supported
- 🧠 Basic movement logic for all 6 piece types
- 🎨 Classic green and white chessboard design

## Pieces & Movement 🏰

| Piece | Movement |
|-------|----------|
| ♙ Pawn | Forward one square, two on first move, diagonal captures |
| ♘ Knight | L-shaped jumps |
| ♗ Bishop | Diagonal slides |
| ♖ Rook | Horizontal and vertical slides |
| ♛ Queen | All directions (bishop + rook combined) |
| ♔ King | One square in any direction |

## Note 📝

This is a basic implementation of chess. Advanced rules like en passant, castling, pawn promotion, and check/checkmate detection are not yet implemented.

## Technologies Used 🛠️

- HTML5
- CSS3
- Vanilla JavaScript
